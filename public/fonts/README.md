# Fonts — PP Right Grotesk

Plaats hier de lokale fontbestanden. De app verwacht deze namen
(zie de `@font-face` regels in `src/index.css`):

- `PPRightGrotesk-WideMedium.woff2` (+ `.woff`)  → titels, logo, actieve componenten
- `PPRightGrotesk-Medium.woff2`     (+ `.woff`)  → knoppen, actielabels, ETA
- `PPRightGrotesk-Light.woff2`      (+ `.woff`)  → body, adressen, tijden

Zolang de bestanden ontbreken valt de app netjes terug op `system-ui`,
dus de app draait ook zonder de fonts.
